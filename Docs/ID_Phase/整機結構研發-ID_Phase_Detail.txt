整機結構研發-ID Phase

A. 阶段性工作描述
	A.1.根据ID和堆叠评估整机结构工艺可行性
		輸入：
			1.CMF工艺说明文件； 
			2.外观分件3D；
			3.堆叠3D；
			4.堆叠设计说明书；
			5.机电元器件规格书及定制器件设计图
			6.产品定义
			7.机电料清单
			8.堆叠 CHECKLIST

	A.2.客户标准可行性评估
		輸入：
			1.客户标准

	A.3.填写ID checklist
		輸入：無

	A.4.针对特殊材质、工艺和结构形式，外发给供应商评估
		輸入：
			 1.供应商评估工艺可实现性报告

	A.5.TP和LCM可行性评估,机电件规格书确认
		輸入：
			1.TP/LCM图纸，机电件规格书

	A.6.装配可行性评估
		輸入：無

	A.7.射频可行性评估
		輸入：
			1.天线可行性报告或邮件

	A.8.ESD可行性评估
		輸入：
			1.ESD可行性评估报告或邮件

	A.9.音频可行性评估（喇叭，听筒，麦克）
		輸入：
			1.音频可行性报告或邮件

	A.10.散热膜位置评估：与硬件沟通散热膜大概位置，评估是否影响整机厚度及装配可行性
		輸入：
			1.相关确认报告或邮件

	A.11.光学可行性评估：按键灯、闪光灯、指示灯、显示屏等
		輸入：
			1.相关确认报告或邮件

B. ID Phase的輸出

		例：XXXID设计评审报告

	2.《XXX结构风险管理表》
		例：XXX结构风险管理表

	3.《ID checklist》
		例：XXX_XXXX_XXX-ID-CHECKLIST
